# Doinu 🎧🎸

**Doinu** is a Free and Open Source (FOSS), ad-free, and ethical music streaming application built with **Flutter**. It is designed specifically to support independent bands and artists by providing a fair and direct space to share their music.

Doinu is developed and maintained by **[TheHiveMinds.in](https://thehiveminds.in)** and is a fork of the **Hivefy Open Source Project**.

[<img src="https://upload.wikimedia.org/wikipedia/commons/7/78/Google_Play_Store_badge_EN.svg" height="40">](https://play.google.com/store/apps/details?id=com.hiveminds.doinu)
[<img src="https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg" height="50">](https://apps.apple.com/us/app/doinu-discover-fresh-music/id6760999984)


## 🌟 Key Features

- **Ethical Streaming**: Every track on Doinu is legally licensed with direct permission from rights holders.
- **Artist-Centric**: Shared directly by bands to ensure a fair and direct connection with listeners.
- **Privacy First**: No ads, no tracking, and no subscriptions required.
- **Offline Ready**: Download your favorite tracks and listen anywhere.
- **Multi-Platform**: Available for Android and iOS.
- **Open Source**: Fully FOSS under the GPL-3.0 license.

## 🚀 Getting Started

### Prerequisites

- **Flutter SDK**: `>=3.10.4`
- **Java JDK**: `17+`
- **Development Environment**: Android Studio, VS Code, or IntelliJ with Flutter/Dart plugins.

### Setup & Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/acermua/doinu
   cd doinu
   ```

2. **Install dependencies**:

   ```bash
   flutter pub get
   ```

3. **Run the application**:
   ```bash
   flutter run
   ```

## 🏗️ Tech Stack

Doinu leverages modern technologies to provide a high-fidelity and minimalist experience:

| Category             | Technology                                          |
| -------------------- | --------------------------------------------------- |
| **Framework**        | Flutter                                             |
| **State Management** | Riverpod                                            |
| **Audio Engine**     | Just Audio & Audio Service                          |
| **Database**         | Drift / Hive (for offline caching)                  |
| **Backend**          | Supabase                                            |
| **Localization**     | Custom i18n supporting English, Spanish, and Basque |
